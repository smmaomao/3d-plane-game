# 3d-plane-game
我想做一个3d前端项目，玩法为打飞机，竖版9:16，闯关游戏，先暂定3关，每关3条命，自己的飞机从下往上飞行，每关时间控制在30-60s，每关最后都有个大boss，中间会有随机的道具可以吃，有“散单、辅助导弹、5束激光” 这样3种道具，道具持续时间为5秒
前端 3D 项目： 引入 Three.js、React Three Fiber、WebGL、Babylon.js 等前端 3D 项目，测试模型对空间坐标、材质、光影、相机视角的理解代码生成能力。  
 ◦  使用 Three.js / React Three Fiber / Babylon.js / WebGL 之一
